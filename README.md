# 🎲 Dice Roller Simulation

A simple dice rolling game made with Python and Pygame. Press **Enter** to roll the dice and receive a random result instantly.

This project was created as a quick and fun simulation for party games, friendly bets, board games, and any situation where you need a random dice roll.

## Features

- Random dice rolling system
- Simple and easy-to-use controls
- Lightweight and beginner-friendly
- Built using Python and Pygame
- Great for learning basic game development concepts

## Requirements

Before running the project, make sure you have Python installed.
